# pds019_485_010

В этом репозитории содержатся исходники проекта `pds019_485_010`. Проект включает в себя две платы: A и B.

На плате А располагается модуль esp32, разъемы для его программирования две кнопки и несколько светодиодов.

На плате B располагается система питания устройства, состоящая из AC-DC модуля, создающего шину питания 12в и DC-DC конвертера 12в -> 3.3в. Кроме этого на плате B располагается гальванически изолированный RS485 интерфейс, реле и разъем для подключения датчика.
